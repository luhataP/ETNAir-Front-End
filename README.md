### ETNAir

ETNAir is a AirBnB-like website made as a marked project by 5 ETNA students (3 developers & 2 network administrators) used with TypeScript and React + Vite.
It allows users to rent and book properties, manage their reservations, and communicate with hosts.

#### Features

- User authentication and profile management
- Property listing and search functionality
- Booking and reservation system
- Communication platform between users and hosts
- Responsive design for mobile and desktop users

#### Installation

1. Clone the repository:
```bash
   git clone
```

2. Navigate to the project directory:
```bash
    cd ETNAir/front
```
3. Install dependencies:
```bash
    npm install
```

4. Start the development server:
```bash
    npm run dev
```

5. Open your browser and go to `http://localhost:5173` to view the application.

#### Technologies Used
- TypeScript
- React
- Vite


#### Website overview explanation

- **Home Page**: The landing page showcases featured properties and provides a comprehensive search interface. Users can filter listings by location, check-in/check-out dates, and number of guests. The page includes property ratings, pricing information, and quick-access filters for popular destinations.

- **Reservation System**: A dedicated section where users can view, manage, and track their bookings. Features include reservation history, cancellation options, payment status, and booking confirmations. Users receive notifications for reservation updates and can modify dates if availability permits.

- **Destinations Page**: Curated collection of popular travel destinations with handpicked property listings. Each destination includes regional information, highlights, seasonal availability, and trending properties to inspire user bookings.

- **Host Page**: Comprehensive dashboard for property owners to list new properties, set pricing and availability, manage guest bookings, and track earnings. Hosts can upload property photos, write descriptions, and set house rules.

- **Interaction Page**: Messaging platform enabling real-time communication between guests and hosts. Users can ask questions about properties, negotiate terms, and receive host responses before booking confirmation.

- **Notice Page**: Information hub displaying important announcements, platform updates, policy changes, and service notifications relevant to users.

- **Help Page**: Support resource center with FAQs, troubleshooting guides, contact forms, and documentation to assist users with technical issues, account management, and platform usage questions.

- **Login Page**: Secure authentication interface for user sign-in and account creation. Supports account registration, password recovery, and optional social media login integration for streamlined access.
